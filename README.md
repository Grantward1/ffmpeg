Useful scripts for ffmpeg

## Usage
1. Clone this repo
2. Initialize submodules:
    ```bash
    git submodule init
    git submodule update
    ```
3. (optional) Add this directory to the path
4. Use scripts

## Included Scripts
Each script should be fairly self explanatory to use. Run `<script.sh> --help` for information on that script's command line arguments.
